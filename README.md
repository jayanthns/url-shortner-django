## Getting started

#### Create virtual environment
`python3 -m venv env`

#### Activate the environment
`source env/bin/activate`


```cd <project_root>```

#### Create migrations
`python manage.py makemigrations`

#### Applying the migrations
`python manage.py migrate`

#### Running the project
`python manage.py runserver`
